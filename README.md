# Gestor de Finanzas Personales

Aplicación web para el control de finanzas personales desarrollada con React y ASP.NET Core 8.

## Integrantes
- Kenya Joseline Aguirre Leyva (Frontend Developer)
- Ariel Jared Agramon Vargas (Backend Developer)

## Tecnologías Utilizadas

### Frontend
- React 18
- React Router DOM para navegación
- Axios para peticiones HTTP
- Chart.js y React-Chartjs-2 para gráficos
- React Icons para iconografía
- CSS Modules para estilos

### Backend
- ASP.NET Core 8
- API RESTful
- CORS configurado
- Swagger/OpenAPI
